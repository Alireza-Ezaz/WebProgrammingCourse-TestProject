# Web Programming HW1 - Multimedia Project
## _Dithering_



[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

Dithering is used in computer graphics to create the illusion of color depth in images on systems with a limited color palette.


## Features

- Import a image and get the dithered version of it.
- specify the scale of dithering
